Sobre mi

Soy Portimagic, QA Automation engeneer. La idea de este blog es usarlo para compartir mi experecien aprendiendo desarrollo web para motivar y ayudar a quienes quieran recorrer el mismo camino. El objetivo es crear un sitio igual a este (creado con wix) y mostrar todo el preoceso. Con cada avance voy a crear una entrada de blog mostrando la evolucion del sitio y contando un poco como fue el proceso para lograrlo. Para ver el sitio, una vez haya algun avance voy a dejar links a la página y al repositorio en git
